# Instructions

This Project is a Python CLI for Telegram. 

## 0x00. Importance

All changes generated by Large Language Models (LLMs) MUST be reviewed and approved by a **human** before being committed. The commit message MUST include an explicit confirmation of this review.

Example:
```text
**Add Documents**

This commit generated by AI. All changes authored by me.

Cline (planned by GPT-5, executed by Gemini 2.5 Pro)
Prompt:
Your job is trying to add documents in this swift package project.
- You should use Apple DocC format;
- You should add README.md;
- You should add doc for every swift file in @/Sources  ;
```


## 0x01. Tech Stack

- Python 3.14
- Package Management: [uv](https://docs.astral.sh/uv/)
- CLI Framework: [Typer](https://typer.tiangolo.com)
- Async Runtime: [asyncio](https://docs.python.org/3/library/asyncio.html)
- Core Framework: [telethon]](https://docs.telethon.dev/en/stable/)
- Other Devlopment Framework: tests: pytest + pytest-asyncio; lint/format: ruff; types: pyright + ty

## 0x01. Source code

- `README.md` the basic inforamtion about this project.
- `AGENTS.md` the instructions for agents like codex.
- `docs` the sub doc file, the most them are used for agents.
- `skills` built-in [agent skills](https://agentskills.io) for quick agent integration. 
- `src` the source code 
  - `tele_cli` the main module
  - `tele_cli/cli` the cli entrypoint


## 0x02. Versioning

The project follows the [Calendar Versioning](https://calver.org/) scheme (`YYYY.MM.PATCH`) .

Examples:
- 2025.12.2
- 2026.1.1
